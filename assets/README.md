# assets/

Drop your custom PNG avatar here.

Requirements:
- Format: PNG (transparent background recommended — like an anime cutout)
- Recommended size: 200×200px or larger
- Filename: anything, e.g. `myavatar.png`

Then in your `.env` file, add:
```
AVATAR_FILE=myavatar.png
```

The image will appear as a round avatar in the top-left of the dashboard.
